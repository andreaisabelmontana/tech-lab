# tech-lab

Low-code, no-code & generative AI, made interactive. 11 self-contained demos, one per idea from the IE BCSAI *Technology with Impact* course (*Low Code, No Code and Generative AI*).

**Live:** https://andreaisabelmontana.github.io/tech-lab/

Trends & impact
1. Technology adoption — tune the Bass diffusion model (`p` innovation, `q` imitation) and watch the S-curve form with Rogers' adopter bands (innovators → laggards)
2. Network effects — compare Metcalfe (`n²`), Sarnoff (`n`) and Reed (`2ⁿ`) value laws on a live network and bar chart

No-code & automation
3. No-code app builder — drop UI/logic blocks (input, button, list, auth, database, API) and see the estimated lines of code saved
4. Automation — build a Zapier/Make trigger → filter → action pipeline and compute hours of manual work saved per month
5. RPA & automation ROI — score a task by volume, time and cost; get the payback period for a bot with a breakeven chart

Generative AI
6. Tokenizer — type text and watch a simplified sub-word tokenizer split it into tokens with IDs, chars/token ratio and a cost estimate
7. Attention & next-token prediction — a toy transformer: attention weights over context tokens feed a temperature-controlled softmax over candidate continuations
8. Diffusion — animate denoising from pure noise toward a prompt target, step by step
9. Chatbot — keyword intent routing (greeting / hours / pricing / support / goodbye) with confidence scores and a fallback

Data & decisions
10. No-code database — filter an Airtable-style table with field / operator / value, mirroring a `WHERE` clause
11. Technology with impact — an impact × feasibility 2×2 matrix that prioritises candidate tech-for-good solutions

Plain HTML + canvas + KaTeX. Indigo accent. Zero build step.
